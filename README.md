# piatra_harite_foarfeca
„Piatră, Hârtie, Foarfecă” jucat împotriva calculatorului. Tu alegi o variantă, el alege una la întâmplare, iar programul decide cine a câștigat runda. Partea mișto e că ține scorul pentru tine și, când te plictisești și apeși „Q”, îți arată bilanțul final: de câte ori l-ai bătut și de câte ori te-a învins el.
